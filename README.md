## 1. Overview and Problem Solved
### Problem Statement
Helps individuals track and manage their finances by providing clear visibility into expenses and income.

### Solution
A Python command-line utility designed to log, save, and summarize financial transactions.

---

## 2. Core Features (The Plan)
* **Transaction Logging:** Allows input of date, category, type (Income/Expense), and amount.
* **Persistent Data:** Transactions are saved to a file (`transactions.txt`).
* **Summary Report:** Calculates the **Net Balance** and provides an **Expense Breakdown** by category.
